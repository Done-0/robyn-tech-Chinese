### [安装 Robyn](https://robyn.tech/documentation/api_reference#installing-robyn)

Robyn 是一个 Python 库，您可以通过 `pip` 或 `conda` 来安装它：

```bash
pip install robyn
```

此外，Robyn 还提供了一些扩展功能，您可以根据需要进行安装：

```bash
pip install "robyn[templating]"
```

建议先安装基础包，然后根据实际需求安装扩展模块。
